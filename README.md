# Mi API con FastAPI

Esto es un ejemplo de un readme para mi aplicación.

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/alangajardomedina/ejemplo_fastapi.git
```

2. Entrar en la carpeta del proyecto:
```bash
cd proyecto_fastapi
```

3. Crear ambiente virtual (venv):
```bash
python -m venv venv
```

4. Activar ambiente virtual:
```bash
source venc/Scripts/activate
```

5. Instalar dependencias que estan en el archivo requirements:
```bash
pip install -r requirements.txt
```

6. Levantar api:
```bash
uvicorn app.main:api --reload
```
